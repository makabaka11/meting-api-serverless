欢迎访问Retr0的音乐API 👋

这是一个音乐api接口，内置了网易云的VIP（因为我充了）
不过只有通过www.retr0.xyz域名才能请求到VIP歌曲

📖 API 使用说明

基础路径: https://meting.retr0.xyz/api

请求参数

参数 必填 说明
server 是 平台: netease (网易云) / tencent (QQ) / kugou (酷狗) / baidu (百度) / kuwo (酷我)
type 是 类型: search / song / album / artist / playlist / lrc / url / pic
id 是 资源 ID (如歌曲 ID、歌单 ID)

示例

  • 获取网易云歌曲详情: https://meting.retr0.xyz/api?server=netease&type=song&id=186016

前端接入 (MetingJS)

如果你使用 MetingJS，只需设置 meting_api 即可:

<meting-js
    server="netease"
    type="playlist"
    id="60198"
    meting_api="https://meting.retr0.xyz/api?server=:server&type=:type&id=:id">
  </meting-js>
